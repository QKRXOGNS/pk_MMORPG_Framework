## 🔧 3. 코드 변경 사항

### `game-client/src/components/GameCanvas.tsx`

#### loadImages() 함수 (118-153줄)
- **변경 전**: 실제 스프라이트 이미지 로딩
- **변경 후**: 주석 처리되고 플레이스홀더로 변경
- **설명**: 스프라이트가 없어도 게임이 작동하도록 함
- **재활성화**: 스프라이트 추가 후 주석 해제 가능

#### drawEntity() 함수 (561-734줄)
- **변경 전**: 스프라이트 이미지로 캐릭터 렌더링
- **변경 후**: 색상 박스로 캐릭터 렌더링
- **기능**:
  - 플레이어: 직업별 색상 박스 (전사=파랑, 궁수=주황, 마법사=보라, 가디언=초록)
  - 몬스터: 타입별 색상 박스 (빨강=공격형, 초록=패시브)
  - 상태 표시: 테두리 색상으로 공격/사망 상태 표시
  - HP 바 및 이름 표시 유지

- **재활성화**: 주석으로 스프라이트 렌더링 코드 제공 (686-734줄)
